#### 题外

搞定了 Gitbook，准备借助此工具，将 Series 65 笔记开源，省得老是和别人共享 Evernote 文件夹，此方式不优雅。  

步骤：  
1. Gitbook 本地初始化；
- 在 Github 创建远程仓库;
- 将本地 Gitbook 仓库 Push 到 Github ;
- 在 Gitbook.io 将 Github 仓库同步过来。
